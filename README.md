# To-Do-List
🛠 Main Features:
1.	Adding Tasks: You enter the content (e.g., "Buy coffee" or "Write code"), and the app stores it in its memory.
2.	Displaying the List: The program numbers your tasks and shows them in a clear way, so you won't forget anything.
3.	Removing Completed Items: Once you finish a task, just enter its number, and the app "throws it in the trash" and clears it from the list.
4.	Interaction: The program "talks" to you – it asks you to choose an option, confirms actions, or lets you know if you've entered something that doesn't make sense.
